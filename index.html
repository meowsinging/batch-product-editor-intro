Spotlight 06HTML 編輯器 + 免費翻譯一條龍
09 / 22
你不可能手寫的格式，
系統幫你寫好。
各平台的商品說明欄位，底層要求的是 HTML 語法——一般人看不懂，更不可能手打。系統整合富文字編輯器，像打 Word 一樣編輯，背後自動轉成正確 HTML。
ABILITY · 01 · 文字樣式自由調整

字體大小、顏色、粗體、對齊——全部可視化。

ABILITY · 02 · 品牌色自動切換

電商2綠色主題、電商1預設色；輸出自動統一，不會貼錯。

ABILITY · 03 · 日文 / 簡中 翻譯 API

一鍵翻譯整份商品說明，完全不花錢——用免費額度。
